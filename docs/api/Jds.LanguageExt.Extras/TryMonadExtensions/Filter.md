# TryMonadExtensions.Filter&lt;TSuccess&gt; method

Returns a Try which filters the result of *tryDelegate*. When *filter* returns `false`, *onFalse* creates the failure result.

```csharp
public static Try<TSuccess> Filter<TSuccess>(this Try<TSuccess> tryDelegate, 
    Func<TSuccess, bool> filter, Func<TSuccess, Exception> onFalse)
```

| parameter | description |
| --- | --- |
| TSuccess | A success type. |
| tryDelegate | A Try. |
| filter | A filtering function. If true, value continues as-is. If false, value is converted to an Exception with *onFalse*. |
| onFalse | A function to convert *TSuccess* to an Exception. |

## Return Value

A Try. If *filter* returns true, current *TSuccess* is returned. If it returns false, the converted Exception is returned.
